<h1>변수</h1>

* 변경 될 수 있는 값
* 하나의 값을 저장하기 위한 수단  
* 어떤 정보에 이름을 붙여서 정보를 저장
* 변수명은 유일하게 만들어줘야 함
* 예약어는 변수명으로 사용할 수 없음 

<h2>변수 선언</h2>

* 값을 저장하기 위한 메모리 공간을 확보
* 변수 사용을 하려면 반드시 선언이 필요
* let name; // 변수 선언문, 값은 할당되지 않음 
* name = 'kim'; // 값의 할당

<h4>let</h4>

* 값을 변경할 수 있음
* 최초 변수에 let을 사용해서 이미 있는 변수명인지 확인(여러사람과 개발할 때 용이함)

<h4>const</h4>

* 절대로 변하지 않는 상수값
* 대문자로 선언
